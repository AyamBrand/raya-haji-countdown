# ☪️ Countdown Hari Raya Aidiladha 1447

> **27 Mei 2026 • 10 Zulhijjah 1447**

Laman web countdown interaktif ke Hari Raya Aidiladha 1447H — lengkap dengan kiraan hari, jam, minit dan saat secara masa nyata.

🔗 **Live demo:** [Klik di sini](https://AyamBrand.github.io/raya-haji-countdown)

---

## ✨ Ciri-ciri

- ⏱️ Countdown masa nyata (hari / jam / minit / saat)
- 🌙 Animasi bulan dan bintang berkelip
- 📱 Boleh dipasang sebagai apps di Android & iOS (PWA)
- 🌐 Boleh digunakan tanpa internet selepas install
- 📖 Tulisan Arab & suasana tema malam emas

---

## 📲 Cara Install Sebagai Apps (Android)

1. Buka laman web ini menggunakan **Google Chrome**
2. Tunggu beberapa saat — akan keluar butang **"📲 Install Sebagai Apps"** di bawah skrin
3. Tekan butang tersebut → **Install**
4. Selesai! Ikon akan muncul di skrin utama telefon anda
